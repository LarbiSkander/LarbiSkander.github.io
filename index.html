Algorithme — My Programming Language

A language I designed and built from scratch — French-syntax keywords, Python-powered lexer, full type system.

What is Algorithme?

v0.1 · open source

Algorithme is a programming language I created entirely from scratch. It uses French keywords — making it natural for Francophone learners — and is powered by a handwritten lexer in Python that breaks source code into tokens the interpreter can understand.

The language supports the full set of programming constructs: variables, arithmetic, input/output, loops (pour, tant que, repeter/jusqua), conditionals (si/sinon/fin_si), functions with return types, procedures with pass-by-reference parameters (@), and typed arrays.

LOOPS
pour · tant que · repeter
CONDITIONS
si · sinon si · sinon · fin_si
FUNCTIONS
fonction · procedure · retourner
TYPES
entier · reel · chaine · boolean · tableau
exemple.algo
-- Fonction qui retourne la somme
fonction somme(a,b:entier):entier
debut
  retourner a + b
fin

-- Procedure avec passage par reference
procedure saisir(@x:entier)
debut
  repeter
    lire(x)
  jusqua x mod 2 = 0
fin

-- Programme principal
pour i de 0 a 5 faire
  ecrire("iteration: " + i)
fin_pour

saisir(x)

si x > 0 alors
  ecrire("pair et positif")
sinon
  ecrire("pair et negatif")
fin_si

How the Lexer Works

The lexer is the first stage of the Algorithme interpreter. It reads raw source code as a string and converts it into a list of typed tokens using Python's re (regex) module. Each token has a type and a value — the parser then uses these tokens to understand the program's structure.

Token Type Example Description
KEYWORD si, pour, fonction, lire Reserved words of the language
IDENTIFIER x, somme, i, result Variable and function names
NUMBER 0, 42, -7 Integer values
FLOAT 3.14, 0.5 Decimal number values
STRING "bonjour", "iteration: " Text between double quotes
ASSIGN <- Assignment operator
SYMBOL ( ) , : + - * / Operators and punctuation
